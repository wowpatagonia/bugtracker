## Instructivo para reportar un problema (bug)

## 1) Verificá si el problema ya fue reportado
Antes de crear un reporte nuevo, entrá a la sección “Issues” (lista de reportes) y usá el buscador. Escribí palabras relacionadas con el problema (por ejemplo, el nombre del jefe, misión o zona) y presiona ENTER para ver si alguien ya lo informó.

<img width="670" height="365" alt="TYRANNUS" src="https://github.com/user-attachments/assets/fe76782d-f089-49e0-be8b-a6620dc8b803" />

## 2) Si no existe un reporte igual, creá uno nuevo
Hacé clic en “New Issue”

<img width="717" height="300" alt="NEW ISSSUE" src="https://github.com/user-attachments/assets/c9b2d302-3521-40aa-bd3a-60422615abcb" />

## 3) Completá el reporte con un título claro y la información solicitada
Para redactar el reporte recomiendo primero clickear y mirar este [>REPORTE DE MUESTRA<](https://github.com/wowpatagonia/bugtracker/issues/1)

Cuando esté todo listo, presioná el botón verde “Submit new Issue” al final del formulario.
Si querés corregir o agregar información, abrí tu reporte, hacé clic en los tres puntos (menú) y elegí “Edit”.

## Seguimiento de reportes y comentarios
Cualquier persona puede revisar y comentar los reportes, es posible que un miembro del staff le pida mas información para continuar la investigación del bug por lo que es importante que el autor responda a las solicitudes de los comentarios.
